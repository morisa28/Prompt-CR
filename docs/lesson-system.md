# 错题本机制说明

## 定位

错题本是 Prompt-CR 的半自动迭代建议机制，用于记录低分 Prompt 的失败模式，统计高频缺失信息，并生成问题树、评分规则和模板的优化建议。

它不是机器学习系统，不会自动训练模型，也不会自动改写规则。

## 数据结构

每条 `PromptMistake` 包含：

- 原始 Prompt；
- 生成 Prompt；
- 生成前后得分；
- 低分维度；
- 错误类型；
- 修复建议；
- 是否建议更新问题树、评分规则或模板。

默认数据文件：

```text
data/prompt-mistakes.json
```

## 统计结果

`MistakeSummary` 会输出：

- 总记录数；
- 按场景统计；
- 按低分维度统计；
- 按错误类型统计；
- 高频缺失信息；
- 问题树、评分规则和模板建议。

## 迭代建议规则

- `acceptanceCriteria` 高频低分：建议强化 Level 4 验证命令和成功标准追问。
- `constraints` 高频低分：建议强化禁止事项和修改范围模板。
- `locatableInputs` 高频低分：建议在 Level 2 增加相关文件路径示例。
- `context` 高频低分：建议在 Web UI 中把工作目录和技术栈置顶。
- 某个场景低分集中：建议增加该场景专属问题。
