# 项目定位

CodePrompt Coach 是一个面向代码生成任务的需求工程辅助系统。它通过分级提问帮助用户澄清目标、上下文、约束和验收标准，再用量化评分诊断 prompt 质量，最后生成适配 coding agent 的高质量任务 prompt。

项目不定位为普通 prompt 网站，也不定位为聊天壳子。它强调“AI 引导用户理解需求”，目标是提高用户的软件工程表达能力和 coding agent 的任务执行稳定性。

## 目标用户

- 计算机相关专业学生。
- 初级开发者。
- 使用 Codex、Claude Code、Gemini CLI、Cursor、Copilot 等 coding agent 的用户。
- 不清楚如何表达开发需求、测试要求、验收标准和边界条件的编程学习者。

## MVP 闭环

```text
模糊需求 -> 分级提问 -> 结构化需求 -> 初始评分 -> prompt 生成 -> 生成后评分 -> 自我审查 -> 错题本 -> 规则/模板迭代
```

## 非 MVP 范围

医疗、法律、金融、营销、创意、多模态、泛文档研究等旧领域已归档，不作为当前大创原型核心方向。
