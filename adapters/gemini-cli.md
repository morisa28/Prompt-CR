# Gemini CLI Adapter

## 1. 适用场景

- 通过 CLI 读取仓库、运行命令、生成代码或报告。
- 需要明确终端命令、路径、环境假设和日志记录的任务。

## 2. 不适用场景

- 需要未授权登录或访问私有外部系统。
- 缺少输入材料且目标无法通过本地文件发现的任务。

## 3. Prompt 构造注意事项

- 明确工作目录、输入路径、命令执行顺序和失败处理。
- 要求把关键命令和结果写入最终报告。
- 对数据或文档任务，要求列出读取材料和未读取材料。

## 4. 必须包含的约束

- 不得声称运行了未运行的命令。
- 不得编造文件、数据、引用或命令输出。
- 破坏性命令、联网安装、生产数据操作必须被排除或要求人工确认。

## 5. 推荐输出格式

```text
读取范围：
执行命令：
结果摘要：
生成/修改文件：
验证状态：
阻塞项：
```

## 6. 常见失败模式

- 没有记录命令输出，最终报告不可追溯。
- 对缺失输入自行猜测。
- 没有区分 CLI 能力和用户未授权范围。
